# JJJTechAviation
If you're new to github, to download this modpack, click the green "code" button, and then click "download zip"
![image](https://raw.githubusercontent.com/jurassicjordan/JJJTechAviation/refs/heads/main/downloadinstructions.png)

To install into prismlauncher click "Add instance" then select the "import option, and select the zip you downloaded

![image2](https://raw.githubusercontent.com/jurassicjordan/JJJTechAviation/refs/heads/main/installinstructions.png)
